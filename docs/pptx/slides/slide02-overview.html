项目概述

项目定位

仿腾讯元宝的 AI 对话 APP，集成 14+ 功能模块，覆盖 AI 对话、图像创作、教育工具、多媒体、效率工具五大领域。

技术栈

React Native NestJS Python TypeScript PostgreSQL ChromaDB LangChain Stable Diffusion

核心能力

AI 对话、角色扮演、图像生成/编辑、教育辅导、语音交互、知识库管理、联网搜索

三端架构

前端 - React Native (Expo Web)

25 个页面 | 14 个功能入口 | Zustand 状态管理

后端 - NestJS

22 个模块 | 60+ API | SSE 流式通信

AI 服务 - Python FastAPI

Stable Diffusion | PaddleOCR | DashScope ASR
