AI 图像创作

AI 生图（Stable Diffusion）

本地 SD 1.5 推理，适配 4GB 显存。float16 + attention slicing 优化。

前端 → POST /create/ai-gen → Python Service (localhost:8000)

→ SD img2img 推理 → 每步回调进度 → 返回图片

智能 P 图

6 种编辑功能：扩图、变清晰、人像美颜、去除杂物、图片美化、风格迁移。复用 SD img2img 管线。

王者 COS 照

人脸融合技术，5 个英雄角色，实时预览生成效果。用户上传人脸 + 选择英雄 → SD 生成 COS 照。

图生图

基于参考图 + Prompt 的图像变换，支持自定义风格和参数调节。

model = StableDiffusionPipeline.from_pretrained("runwayml/stable-diffusion-v1-5")
