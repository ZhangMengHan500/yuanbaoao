AI 教育工具

拍题答疑

DashScope 视觉模型端到端识别题目 + 解题。不使用 OCR，直接用视觉大模型处理。

技术亮点：qwen-vl-plus 视觉模型，一次调用完成识别+解题

作业批改

视觉模型识别手写答案，逐题批改。输出：总体评价 → 逐题批改 → 知识点总结 → 学习建议。

AI 出卷

三种模式：相似卷（同知识点相似题）、错题巩固卷（针对性练习）、自定义试卷（自由描述）。

流程：视觉模型识别题目 JSON → LLM 生成试卷 → SSE 流式返回

文档阅读（RAG 管线）

PDF/TXT 解析 → 分块(512字符) → ChromaDB 向量化 → AI 摘要(执行摘要+要点+大纲) → 文档问答(引用溯源)。
